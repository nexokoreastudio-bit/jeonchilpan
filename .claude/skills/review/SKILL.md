## 코드 리뷰

현재 git 변경사항을 분석하고 리뷰합니다.

### 분석 항목
1. `git diff --stat` — 변경 파일 요약
2. `git diff` — 전체 diff 읽기
3. 각 변경에 대해:
   - 의도 파악 (기능 추가/수정/삭제)
   - 잠재적 버그 확인
   - 보안 이슈 (XSS, SQL injection 등)
   - 타입 안전성
   - 성능 이슈
   - 코드 스타일 일관성

### 출력 형식
- 파일별 변경 요약
- 발견된 이슈 (심각도: 높음/중간/낮음)
- 개선 제안
- 커밋 메시지 초안 제안

커밋되지 않은 변경사항이 없으면 최근 커밋 1개를 리뷰.
