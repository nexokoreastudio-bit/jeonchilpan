# 소상공인 스마트상점 모집 캠페인 명세 (전칠판)

## 1) 목적
- 전칠판 메인에서 `소상공인 스마트상점 모집` 캠페인 노출 강화
- 방문자에게 빠르게 자격 확인/상담 신청 행동 유도
- 기존 전환 경로(`시연 신청`, `견적 요청`)와 자연스럽게 연결

## 2) 참조 프로젝트
- 레퍼런스: `../nexo-policy-fund-landing`
- 참조 요소:
  - 히어로 중심 배치(강한 헤드라인 + 행동 버튼)
  - 팝업형 공지/안내 모달 패턴

## 3) 구현 범위

### A. 메인 히어로 섹션 (홈)
- 위치: 메인 페이지 상단 콘텐츠 영역
- 메시지:
  - 제목: 소상공인 스마트상점 모집
  - 보조문구: 대상/혜택/신청절차 요약
- CTA:
  - `모집 안내 보기` -> `/smartstore`
  - `신청 상담받기` -> `/leads/quote?campaign=smartstore`

### B. 홈 팝업 배너
- 위치: 홈 진입 시 중앙 모달
- 기본 동작:
  - 홈 최초 진입 시 노출
  - `오늘 하루 보지 않기` 선택 시 24시간 숨김(localStorage)
- CTA:
  - `모집 안내 보기` -> `/smartstore`
  - `견적 상담` -> `/leads/quote?campaign=smartstore_popup`

### C. 캠페인 상세 페이지
- 경로: `/smartstore`
- 섹션:
  - 모집 핵심 요약
  - 지원 대상
  - 신청 절차 3단계
  - CTA 2종(시연/견적)

### D. 메뉴 노출
- 상단 메뉴 드롭다운에 `스마트상점 모집` 링크 추가 (`/smartstore`)

## 4) 카피 가이드
- 과장 표현보다 실행 정보 중심
- 필수 문구:
  - 대상/혜택/절차
  - 상담 전환 CTA
- 운영 단계에서 신청 기간/정책 수치 업데이트 가능하도록 구조 단순화

## 5) 측정 포인트(후속)
- 히어로 CTA 클릭률
- 팝업 CTA 클릭률
- `/smartstore` -> 견적/시연 전환율
